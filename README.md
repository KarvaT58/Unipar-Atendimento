# Unipar Atendimento

Aplicação web **Unipar** para atendimento e dashboard, construída com [Next.js](https://nextjs.org), [React](https://react.dev) e [shadcn/ui](https://ui.shadcn.com).

## Desenvolvimento

```bash
npm install
npm run dev
```

Abra [http://localhost:3000](http://localhost:3000).

## Scripts

- `npm run dev` — servidor de desenvolvimento (Turbopack)
- `npm run build` — build de produção
- `npm run lint` — ESLint
- `npm run test` — Vitest

## Componentes shadcn

Para adicionar componentes:

```bash
npx shadcn@latest add button
```

Os arquivos vão para `components/ui/`.
